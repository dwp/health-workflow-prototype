Strategical Status Iteration

Status iteration miro board

Status alignment on Confluence

This prototype looks at status iteration from what we have learnt so far at Tresco House. It allows a user to do the following:

  • Assessment booked page in alphabetical order
  • Time stamp on last action, replayed to user in 'XX days ago' format
  • Status descriptions removed
  • Conditional reveal for sub-status (gathering information, case manager review and awaiting evidence on the update status page).
  • Sub-status displayed on detail page
